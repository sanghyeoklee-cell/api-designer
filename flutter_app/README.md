# API Designer — desktop app

This is the Flutter desktop shell for **API Designer**. See the project root
[`README.md`](../README.md) for what the project is, setup, usage, and the
responsible-use notice. ([한국어](../README.ko.md) · [中文](../README.zh.md))

Quick start:

```bash
flutter pub get
flutter run -d macos
```

The app launches and manages the Python backend in `../backend`.
